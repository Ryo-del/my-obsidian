| Оператор | Значение              | Пример            |
| -------- | --------------------- | ----------------- |
| `-eq`    | Равно (equal)         | `[ "$a" -eq 5 ]`  |
| `-ne`    | Не равно (not equal)  | `[ "$a" -ne 10 ]` |
| `-gt`    | Больше (greater than) | `[ "$a" -gt 3 ]`  |
| `-ge`    | Больше или равно      | `[ "$a" -ge 7 ]`  |
| `-lt`    | Меньше (less than)    | `[ "$a" -lt 2 ]`  |
| `-le`    | Меньше или равно      | `[ "$a" -le 8 ]`  |
